Privacy Policy for AI Web Crawler Extension

Data Collection:
- GCP Project IDs and access tokens (stored locally in browser)
- Web page content (processed by Google Vertex AI)

Data Usage:
- Credentials are used solely to authenticate with Google Vertex AI
- Page content is sent to Vertex AI for AI analysis as requested by user
- No data is stored on external servers by this extension

Data Storage:
- All credentials are stored locally in your browser using localStorage
- No data is transmitted to third parties except Google Vertex AI as intended

Data Sharing:
- Page content is shared with Google Vertex AI when you use the extension
- No other data sharing occurs
